# 발표 대본 (1-1부터 5-2까지)

## 1-1: 01-alert (기본 알림)
- `1.html`을 열어 `alert("Hello JavaScript")`로 브라우저 알림창이 뜨는지 보여줍니다.
- 주석을 해제하고 메시지를 본인 이름으로 바꾸며, “자바스크립트가 실행 중임”을 직관적으로 확인합니다.

## 1-2: 02-console (콘솔 출력)
- `1.html`에서 `console.log`로 문자열과 숫자를 찍고, `typeof`로 타입을 확인합니다.
- 개발자도구 콘솔을 열어 “Xin chào”, 숫자 10, 그리고 각 타입이 출력되는지 보여줍니다.
- 콘솔이 디버깅과 학습에 유용한 기본 창구임을 강조합니다.

## 3-1: 03-datatype/1.html (데이터 선언)
- 문자열, 숫자, 불리언 변수를 선언하고 콘솔에 출력합니다.
- 데이터 타입별 기본 값을 소개합니다.

## 3-2: 03-datatype/2.html (기초 연산)
- `+`, `-`, `*`, `/`, `%` 기본 연산 결과를 콘솔에서 확인합니다.
- `a++`, `b--`로 증감 연산이 어떻게 반영되는지 보여줍니다.

## 3-3: 03-datatype/3.html (if 조건문)
- 온도 값에 따라 “hot/warm/cold” 메시지를 다르게 출력합니다.
- 점수에 따라 A~F 학점을 구간별 조건으로 출력하며, `if / else if / else` 흐름을 설명합니다.

## 3-4: 03-datatype/4.html (for 반복문)
- 1부터 5까지 카운트하며 `for` 반복문의 초기값, 조건, 증감식을 설명합니다.
- 배열에 담긴 과일 목록을 인덱스로 순회해 로그를 찍으며 배열 길이(`length`) 활용을 보여줍니다.

## 4-1: 04-click/1.html (클릭 이벤트 기본)
- 버튼을 만들고 `getElementById`로 요소를 찾은 뒤 `onclick`에 함수를 연결합니다.
- 버튼을 누를 때마다 콘솔에 “Button clicked”가 찍히는지 보여주며, DOM 요소에 이벤트를 다는 기본 패턴을 설명합니다.

## 4-2: 04-click/2.html (클릭 횟수 누적)
- `clickCount` 변수를 두고 버튼 클릭마다 증가시켜 콘솔에 “Button clicked n times”를 출력합니다.
- 상태 변수 관리와 이벤트 콜백 내부 로직을 강조합니다.

## 5-1: 05-dom/1.html (DOM 내용 전환)
- 화면의 제목과 문장을 `textContent`로 교체합니다. “Change Content” 클릭 시 새로운 문구로, “Reset” 클릭 시 원래 문구로 되돌립니다.
- `getElementById`로 요소를 가져와 `onclick` 이벤트에서 DOM 텍스트를 직접 바꾸는 흐름을 보여줍니다.

## 5-2: 05-dom/2.html (DOM CSS 전환)
- `highlight` 클래스를 미리 CSS로 정의해 배경색/글자색을 바꿔둡니다.
- “Toggle Style” 버튼을 누를 때마다 `classList.toggle("highlight")`로 클래스를 적용/해제하며 스타일이 전환되는 것을 보여줍니다.
- 내용 변경(5-1)과 스타일 변경(5-2)이 동일한 이벤트 패턴 위에서 손쉽게 확장된다는 점을 정리하며 마무리합니다.
